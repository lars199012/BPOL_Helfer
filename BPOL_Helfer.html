Anleitung: Datenimport aus Aufenthaltsrechner

1. PDF erstellen

Erstelle eine PDF über den Button Drucken... Wähle als Drucker "als PDF Speichern" oder "Microsoft Print to PDF" aus und nicht "PDF-Xchange Standart".

als PDF Speichern
Microsoft Print to PDF
PDF-Xchange Standart

2. Textauswahl-Werkzeug aktivieren

Öffne die PDF (mit dem Acrobat Reader) und klicke oben links im Reiter "Start" auf den darunterliegenden Button "Textauswahl".

3. Text markieren

Markiere die komplette Berechnung der Aufenthaltsdauer (die komplette Seite).

4. Text kopieren

Kopiere den Text mit dem Tastendruck STRG + C, oder mit einem Rechtsklick + "Kopieren".

5. Text einfügen

Füge den Text in das Textfeld ein, welches sich beim Import öffnet.

Export als Text

Dieser Text kann kopiert und in einer `Vorlagen.js`-Datei gespeichert werden.
